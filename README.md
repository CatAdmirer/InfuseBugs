# InfuseBugs
This is where you can report bugs for the [Infuse SMP](https://modrinth.com/plugin/infusesmp) plugin.  

Head over to the [Issues](https://github.com/CatAdmirer/InfuseBugs/issues) tab and put your problem there.  
Make sure to include the following:
- Server architecture (Paper, Spigot, etc)
- Minecraft version
- A screenshot of the problem
- Your latest.log file (optional if no errors are shown there)

### Notes:
Do not make duplicate issues.  If your issue is a duplicate of another, it will be closed.  
If your version of infuse is not the latest version (currently 2.4.0), your issue will be closed.  
If your version of minecraft is not a version supported by infuse (anything before 1.21.4), your issue will be closed.
